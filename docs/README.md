# Grihini Product README

Grihini is a Nepal-focused home-service booking app where customers can quickly book trusted workers for day-to-day household tasks.

## What It Is

Grihini connects two user groups:

- Customers: Browse workers, compare trust signals, create bookings, track status, and rate completed jobs.
- Workers: Receive booking requests, accept/decline jobs, manage active work status, and maintain service profile details.

## Visual Direction

The interface uses a premium household style to feel warm, safe, and easy to understand.

Theme palette used in app:

- Primary Burgundy: `#7B1E2B`
- Dark Maroon: `#4A0F1A`
- Cream Background: `#FFF6EE`
- Surface Tint: `#F8EDE3`
- Card: `#FFFFFF`
- Border: `#E6D8CC`
- Text: `#2A1F1A`
- Secondary Text: `#6B5B53`
- Accent Gold: `#D4A017`

## Day 1 Key Features

- Customer Home with service categories and top worker cards.
- Worker List with practical filters (category, price, rating, availability).
- Worker Profile with trust badges and review snippets.
- Booking creation flow with date/time, duration, address, and notes.
- Booking Details with status timeline.
- Rate Worker flow with stars + short review.
- Worker Dashboard for active jobs.
- Requests Inbox for accept/decline actions.
- Worker Profile Edit for services, rate, and availability.
- Demo mode switch between Customer and Worker.

## Design Notes

- Trust: Every worker card/profile highlights verification badges, ratings, and completed jobs.
- Clarity: Card layouts and strong section headers reduce cognitive load for first-time users.
- Warmth: Cream surfaces and burgundy accents create a household-friendly, non-corporate tone.
